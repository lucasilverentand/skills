# Git Skills

Skills for Git workflows and collaboration patterns.

## Skills

| Skill | Description |
|-------|-------------|
| [git-branching](./git-branching/) | Branching strategies and workflows |
| [pr-templates](./pr-templates/) | Pull request templates and guidelines |
| [commit-conventions](./commit-conventions/) | Conventional commits and message formatting |
| [merge-conflicts](./merge-conflicts/) | Conflict resolution strategies |

## Usage

```
/git-branching                 # Set up branching strategy
/pr-templates                  # Create PR templates
/commit-conventions            # Configure conventional commits
/merge-conflicts               # Help resolve merge conflicts
```

## Branching Strategies

- **GitHub Flow** - Simple feature branch workflow
- **GitFlow** - Release-based workflow
- **Trunk-Based** - Short-lived branches

## Commit Conventions

- Conventional Commits
- Semantic commit messages
- Changelog generation
- Release automation integration
