# Branching

Branch strategy, create and manage branches, rebase workflows, and branch comparison.

## Responsibilities

- Define and enforce branching strategies
- Create and manage branches
- Handle rebase workflows and branch comparisons
- Clean up merged and stale branches
- Validate branch naming conventions
- Generate branch summary reports

## Tools

- `tools/stale-branches.ts` — list branches with no commits in the last 30 days
- `tools/branch-naming-check.ts` — validate branch names against naming conventions
- `tools/branch-diff-summary.ts` — summarize changes between two branches
