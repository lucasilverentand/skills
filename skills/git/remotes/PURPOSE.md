# Remotes

Manage remote repositories, sync forks, and maintain remote-tracking branches.

## Responsibilities

- Add, remove, and configure remotes
- Sync forked repos with upstream
- Track ahead/behind status for all remotes
- Prune stale remote-tracking branches
- Manage multi-remote workflows for forks and mirrors

## Tools

- `tools/remote-sync.ts` — sync a fork with its upstream remote
