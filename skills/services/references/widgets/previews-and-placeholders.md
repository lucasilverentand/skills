# Previews and placeholders
- **Design a realistic preview to display in the widget gallery.** To enable users to make an informed choice, your widget's capabilities must be highlighted and clearly represented across all available sizes or types. While you may use live data in the preview, if generating or loading that information takes excessive time, display realistic simulated data instead.
- **Design placeholder content that helps people recognize your widget.** When an installed widget is loading data, it should display effective placeholder content. Achieve this by combining static interface elements with semi-transparent shapes that represent dynamic information. For instance, use rectangles of varying widths to suggest lines of text, and employ circles or squares in place of glyphs or images.
- **Write a succinct widget description.** The descriptions shown in the widget gallery help users understand the function of each widget. Begin your description with an action verb—for example, “See the current weather conditions and forecast for a location” or “Keep track of your upcoming events and meetings.” Avoid phrases that unnecessarily reference the widget itself, such as “This widget shows…,” “Use this widget to…,” or “Add this widget.” Use approachable language and [sentence-style capitalization](https://support.apple.com/guide/applestyleguide/c-apsgb744e4a3/web#apdca93e113f1d64).
- **Group your widget’s sizes together, and provide a single description.** If your widget is offered in multiple sizes, group them to prevent users from believing each size represents a unique product. Provide one unified description for your widget, regardless of the number of sizes offered, to prevent repetition and help users understand how each size offers a slightly different view of the same content and functionality.
- **Consider coloring the Add button.** Once users select your app in the widget gallery, an accompanying Add button appears beneath the collection of widgets you offer. You have the option to specify a color for this button to reinforce your brand identity.
