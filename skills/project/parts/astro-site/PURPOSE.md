# Astro Site

Astro marketing and content site setup, pages, and integrations.

## Responsibilities

- Set up Astro sites for marketing and content
- Create pages and layouts
- Configure integrations
- Manage content collections and MDX pipelines
- Set up SEO metadata and Open Graph defaults
- Configure static asset optimization and image handling

## Tools

- `tools/page-list.ts` — list all Astro pages and their routes with layout assignments
- `tools/collection-check.ts` — validate content collection schemas against existing entries
- `tools/integration-audit.ts` — show installed Astro integrations and their configuration status
