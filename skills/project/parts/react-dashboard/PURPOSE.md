# React Dashboard

Admin dashboards and internal tools: TanStack Start, TanStack Router, React 19, Tailwind CSS.

## Responsibilities

- Set up TanStack Start project with routing and server functions
- Create dashboard pages with server-side data loading
- Build data tables with sorting, filtering, and pagination
- Wire up forms with server actions and validation
- Implement auth guards and role-based route protection
- Manage client state with nanostores and server state with TanStack Query

## Tools

- `tools/route-list.ts` — list all dashboard routes with loaders, auth requirements, and component paths
- `tools/page-scaffold.ts` — generate a new page with loader, component, and auth guard boilerplate
