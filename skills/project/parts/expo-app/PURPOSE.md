# Expo App

Expo React Native app setup, conventions, and navigation.

## Responsibilities

- Set up Expo React Native applications
- Establish app conventions and file structure
- Configure navigation with expo-router
- Manage native module configuration in app.config.ts
- Set up NativeWind styling and theme tokens
- Configure deep linking and typed route definitions

## Tools

- `tools/route-map.ts` — list all expo-router screens with their path patterns and params
- `tools/native-check.ts` — verify native module compatibility and Expo config plugins
- `tools/screen-scaffold.ts` — generate a new screen file with layout, loader, and navigation typing
