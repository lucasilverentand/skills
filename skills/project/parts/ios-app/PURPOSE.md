# iOS App

iOS app: Swift 6.2, SwiftUI, SwiftData, Tuist, zero third-party dependencies.

## Responsibilities

- Set up Tuist project with targets and dependencies
- Create SwiftUI views with @Observable view models
- Configure SwiftData models and persistence
- Write tests with Swift Testing framework
- Manage navigation with NavigationStack and typed routing
- Enforce zero-dependency policy and modern concurrency

## Tools

- `tools/target-list.ts` — list all Tuist targets with bundle IDs, deployment targets, and dependencies
- `tools/project-audit.ts` — check project health: Swift version, deployment target, dependency count, test coverage
