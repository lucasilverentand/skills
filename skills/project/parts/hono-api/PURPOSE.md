# Hono API

Hono API service setup, routes, middleware, and auth integration.

## Responsibilities

- Set up Hono API services
- Define routes and middleware
- Integrate authentication
- Configure CORS and rate limiting
- Set up request validation with Zod
- Generate OpenAPI route documentation

## Tools

- `tools/route-list.ts` — list all registered routes with their methods and paths
- `tools/middleware-audit.ts` — show middleware stack for each route group
- `tools/endpoint-scaffold.ts` — generate a new route file with validation and handler boilerplate
