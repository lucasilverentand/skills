# Presentation

## Topics
- **Action sheets** — An action sheet is a modal display used to present options stemming from an action the user begins.
- **Alerts** — An alert provides users with essential information requiring immediate attention.
- **Page controls** — A page control shows a sequence of indicator images, where each image represents a specific page within a linear list.
- **Panels** — Within a macOS application, a panel usually floats over other open windows, offering additional controls, settings, or details pertaining to the currently selected item or active window.
- **Popovers** — A popover is a temporary view that overlays other content after a user clicks or taps an interactive control.
- **Scroll views** — A scroll view allows users to see content that exceeds the boundaries of the view by shifting the content either vertically or horizontally.
- **Sheets** — A sheet assists users in completing a focused task that is directly connected to their present context.
- **Windows** — A window displays the user interface views and components within your application or game.
