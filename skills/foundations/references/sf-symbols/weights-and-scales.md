# Weights and scales
SF Symbols provides symbols in a wide range of weights and scales, enabling you to create highly adaptable designs.

The nine symbol weights available—ranging from ultralight through black—correspond directly to the weights of the San Francisco system font. This correspondence enables precise weight matching between symbols and adjacent text, while also providing flexibility for different sizes and contexts.

Additionally, each symbol is available in three scales: small, medium (the default), and large. These scales are defined relative to the cap height of the San Francisco system font.

By specifying a scale, you can adjust a symbol's visual emphasis compared to nearby text without disrupting the weight matching achieved with text using the identical point size. For developer guidance, please refer to [imageScale(_:)](apple:SwiftUI/View/imageScale(_:)) (SwiftUI), [UIImage.SymbolScale](apple:UIKit/UIImage/SymbolScale) (UIKit), and [NSImage.SymbolConfiguration](apple:AppKit/NSImage/SymbolConfiguration-swift.class) (AppKit).
