# Performance Skills

Skills for application performance analysis and optimization.

## Skills

| Skill | Description |
|-------|-------------|
| [bundle-analysis](./bundle-analysis/) | JavaScript bundle size optimization |
| [lighthouse-audit](./lighthouse-audit/) | Web vitals and Lighthouse improvements |
| [profiling](./profiling/) | Runtime performance profiling |
| [caching-strategies](./caching-strategies/) | Cache implementation patterns |

## Usage

```
/bundle-analysis               # Analyze and optimize bundle size
/lighthouse-audit              # Improve Lighthouse scores
/profiling                     # Profile application performance
/caching-strategies            # Implement caching patterns
```

## Performance Metrics

- **Core Web Vitals** - LCP, FID, CLS
- **Bundle Size** - JavaScript payload
- **TTFB** - Time to first byte
- **Memory** - Heap usage and leaks

## Optimization Techniques

- Code splitting
- Tree shaking
- Lazy loading
- Image optimization
- Edge caching
