# Testing Skills

Skills for testing applications across all layers - unit, integration, and end-to-end.

## Skills

| Skill | Description |
|-------|-------------|
| [playwright-e2e](./playwright-e2e/) | End-to-end browser testing with Playwright |
| [vitest-unit](./vitest-unit/) | Unit testing with Vitest |
| [test-coverage](./test-coverage/) | Coverage analysis and improvement strategies |
| [api-testing](./api-testing/) | REST and GraphQL API testing |
| [snapshot-testing](./snapshot-testing/) | Visual and data snapshot testing |

## Usage

```
/playwright-e2e                # Set up Playwright E2E tests
/vitest-unit                   # Configure Vitest for unit testing
/test-coverage analyze         # Analyze and improve test coverage
/api-testing rest              # Set up REST API tests
```

## Testing Pyramid

- **Unit Tests** - Fast, isolated function/component tests
- **Integration Tests** - Module interaction tests
- **E2E Tests** - Full user flow tests

## Supported Frameworks

- Vitest
- Playwright
- Jest
- Testing Library
- Supertest
