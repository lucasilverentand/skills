# Analytics

Query analytics data, set up dashboards, and configure monitoring views.

## Responsibilities

- Query and analyze analytics data
- Set up dashboards and monitoring views
- Define and track key metrics and conversion funnels
- Generate periodic reports summarizing trends and anomalies

## Tools

- `tools/query-runner.ts` — execute analytics queries against configured data sources and format results as tables
- `tools/funnel-builder.ts` — define multi-step conversion funnels and calculate drop-off rates from event data
- `tools/report-gen.ts` — generate a summary report of key metrics with period-over-period comparisons
