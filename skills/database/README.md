# Database Skills

Skills for database schema design, ORMs, and data management.

## Skills

| Skill | Description |
|-------|-------------|
| [prisma-schema](./prisma-schema/) | Prisma schema design and migrations |
| [drizzle-orm](./drizzle-orm/) | Drizzle ORM setup and query patterns |
| [sql-optimization](./sql-optimization/) | Query analysis and indexing strategies |
| [database-migrations](./database-migrations/) | Migration strategies and rollback patterns |

## Usage

```
/prisma-schema                 # Design Prisma schema
/drizzle-orm                   # Set up Drizzle ORM
/sql-optimization              # Analyze and optimize queries
/database-migrations           # Plan migration strategy
```

## Database Types

- **Relational** - PostgreSQL, MySQL, SQLite
- **Document** - MongoDB
- **Key-Value** - Redis
- **Edge** - Cloudflare D1, Turso

## ORM Support

- Prisma
- Drizzle
- Kysely
- TypeORM
