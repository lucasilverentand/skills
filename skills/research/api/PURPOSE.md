# API Research

Explore external APIs, compare options, plan integrations, and evaluate SDKs.

## Responsibilities

- Explore and evaluate external APIs
- Compare API options and SDKs
- Plan API integrations
- Assess rate limits, pricing tiers, and quota constraints
- Review API authentication schemes and security models
- Evaluate API versioning strategies and deprecation policies

## Tools

- `tools/api-compare.ts` — fetch OpenAPI specs from multiple APIs and generate a side-by-side feature comparison
- `tools/rate-limit-check.ts` — probe API endpoints and summarize rate limit headers and quota policies
- `tools/sdk-size-audit.ts` — measure bundle size and dependency count of candidate SDK packages
