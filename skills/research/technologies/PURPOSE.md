# Technologies

Evaluate technologies, assess migration feasibility, and stay current.

## Responsibilities

- Evaluate new technologies and frameworks
- Assess migration feasibility
- Track technology trends and updates
- Benchmark performance characteristics of candidate technologies
- Analyze community health, maintenance activity, and ecosystem maturity
- Compare licensing models and compliance requirements

## Tools

- `tools/npm-health.ts` — check download trends, open issues, last publish date, and bus factor for npm packages
- `tools/migration-estimator.ts` — scan a codebase for usage of a library and estimate migration effort to a replacement
- `tools/license-audit.ts` — collect and summarize licenses of all transitive dependencies for compliance review
