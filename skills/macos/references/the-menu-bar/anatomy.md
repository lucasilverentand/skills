# Anatomy
When displayed in the menu bar, the menus follow this sequence:

- *YourAppName* (use a brief version of your application's name for this menu title)
- File
- Edit
- Format
- View
- App-specific menus, if applicable
- Window
- Help

Furthermore, the macOS menu bar integrates the Apple menu on the leading side and any additional menu bar items on the trailing side. Refer to [macOS](#macOS) for implementation details.
