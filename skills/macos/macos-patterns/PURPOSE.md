# macOS Patterns

macOS-specific SwiftUI patterns — window management, MenuBarExtra, Settings, toolbars, menu commands, sandboxing, and AppKit interop.

## Responsibilities

- Structure a macOS app with proper Scene types (WindowGroup, Settings, MenuBarExtra)
- Manage multiple windows with openWindow and WindowGroup
- Build toolbars with proper placement and liquid glass integration
- Add keyboard shortcuts and menu bar commands
- Create Settings scenes with proper tab navigation
- Bridge to AppKit with NSViewRepresentable when SwiftUI lacks native support
- Configure entitlements and sandboxing correctly
- Handle file access, drag and drop, and pasteboard
