# Views

SwiftUI view composition, layout, modifiers, animations, and liquid glass design for macOS apps.

## Responsibilities

- Build views using SwiftUI's declarative syntax with proper composition
- Choose the right layout container (VStack, HStack, ZStack, Grid, LazyStacks)
- Apply liquid glass design language correctly (glassEffect, materials, depth)
- Create custom view modifiers for reusable styling
- Implement animations and transitions
- Build custom shapes and paths
- Handle dynamic type and accessibility in views
- Use ViewBuilder for composable container views
