# Data

SwiftData persistence for macOS apps — models, queries, migrations, ModelContainer setup, and UserDefaults.

## Responsibilities

- Design SwiftData models with proper attributes and relationships
- Set up ModelContainer and ModelContext in the app lifecycle
- Write queries with #Predicate and SortDescriptor
- Handle schema migrations for evolving data models
- Use @Query in views for reactive data binding
- Persist simple preferences with UserDefaults and @AppStorage
- Structure model layer for testability with protocol abstractions
- Handle CloudKit sync with SwiftData when needed
