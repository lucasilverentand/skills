# Frontend Skills

Skills for modern frontend development with React and related technologies.

## Skills

| Skill | Description |
|-------|-------------|
| [nextjs-app](./nextjs-app/) | Next.js App Router patterns and setup |
| [react-components](./react-components/) | React component design patterns |
| [tailwind-styling](./tailwind-styling/) | Tailwind CSS utility patterns |
| [state-management](./state-management/) | State management with Zustand, Jotai, Redux |

## Usage

```
/nextjs-app                    # Set up Next.js App Router project
/react-components              # Create React component patterns
/tailwind-styling              # Configure and use Tailwind CSS
/state-management zustand      # Set up Zustand state management
```

## Framework Support

- **React** - Component library
- **Next.js** - Full-stack React framework
- **Remix** - Web standards framework
- **Astro** - Content-focused framework

## Styling Solutions

- Tailwind CSS
- CSS Modules
- Styled Components
- Vanilla Extract
