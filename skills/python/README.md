# Python Skills

Skills for Python web development and tooling.

## Skills

| Skill | Description |
|-------|-------------|
| [fastapi-init](./fastapi-init/) | FastAPI project setup and patterns |
| [django-models](./django-models/) | Django ORM and model design |
| [poetry-setup](./poetry-setup/) | Dependency management with Poetry |

## Usage

```
/fastapi-init                  # Create FastAPI project
/django-models                 # Design Django models
/poetry-setup                  # Initialize Poetry project
/pytest-testing                # Set up pytest configuration
```
