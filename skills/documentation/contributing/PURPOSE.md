# Contributing

Create CONTRIBUTING.md files, PR templates, and code of conduct documents.

## Responsibilities

- Create and maintain CONTRIBUTING.md
- Set up PR and issue templates
- Draft code of conduct documents
- Document branching strategy and commit conventions
- Define development setup and prerequisite instructions

## Tools

- `tools/template-scaffold.ts` — generate GitHub issue and PR templates from project conventions
- `tools/contrib-gen.ts` — scaffold a CONTRIBUTING.md based on repo tooling, test runner, and branch strategy
- `tools/setup-validator.ts` — verify that documented setup steps actually work by running them in a temp directory
