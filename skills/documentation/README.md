# Documentation Skills

Skills for generating and maintaining project documentation.

## Skills

| Skill | Description |
|-------|-------------|
| [readme-generation](./readme-generation/) | Project README creation and templates |
| [api-docs](./api-docs/) | API documentation generation |
| [jsdoc-comments](./jsdoc-comments/) | Code documentation with JSDoc/TSDoc |
| [architecture-diagrams](./architecture-diagrams/) | System design documentation |

## Usage

```
/readme-generation             # Generate project README
/api-docs                      # Generate API documentation
/jsdoc-comments                # Add JSDoc comments to code
/architecture-diagrams         # Create architecture docs
```

## Documentation Types

- **README** - Project overview and quickstart
- **API Reference** - Endpoint/function documentation
- **Guides** - How-to tutorials
- **Architecture** - System design decisions

## Output Formats

- Markdown
- HTML (via static site generators)
- OpenAPI/Swagger UI
- Mermaid diagrams
