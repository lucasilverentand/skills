# Status Report

Writes project status and sprint update reports from git history and project state.

## Responsibilities

- Generate weekly, sprint, and monthly status reports
- Collect and summarize git activity, PR counts, and contributor stats
- Highlight blockers and risks with impact and mitigation
- Track metrics and trends across reporting periods
- Format for different audiences (team, stakeholders, cross-team)
