# Comparison Report

Writes structured comparison reports evaluating multiple options against defined criteria.

## Responsibilities

- Define evaluation criteria with weights before researching options
- Research and document each option's strengths and weaknesses
- Build comparison matrices with consistent scoring
- Make clear, reasoned recommendations with trade-off acknowledgment
- Assess risks of the recommended option with mitigations
