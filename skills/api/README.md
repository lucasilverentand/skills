# API Development Skills

Skills for designing and documenting APIs.

## Skills

| Skill | Description |
|-------|-------------|
| [openapi-spec](./openapi-spec/) | OpenAPI/Swagger specification authoring |
| [graphql-schema](./graphql-schema/) | GraphQL schema and resolver design |
| [rest-design](./rest-design/) | RESTful API design patterns |
| [api-versioning](./api-versioning/) | API version management strategies |

## Usage

```
/openapi-spec                  # Generate OpenAPI specification
/graphql-schema                # Design GraphQL schema
/rest-design                   # Design REST API endpoints
/api-versioning                # Plan API versioning strategy
```

## API Patterns

- **REST** - Resource-based HTTP APIs
- **GraphQL** - Query language for APIs
- **RPC** - Remote procedure call patterns
- **tRPC** - End-to-end typesafe APIs

## Documentation Standards

- OpenAPI 3.0/3.1
- GraphQL SDL
- JSON Schema
- AsyncAPI (for events)
