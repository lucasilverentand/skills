# Scroll views
A scroll view enables users to examine content that exceeds the display boundaries by shifting the content along a vertical or horizontal axis

## Platform guidance — iOS & iPadOS
**Consider displaying a page control when the scroll view operates in discrete, page-by-page mode.** [Page controls](page-controls.md) indicate the total number of pages, screens, or content segments available and specify which segment is currently displayed. For instance, the Weather app utilizes a page control to track transitions between users' saved locations. If both a page control and a scroll view are present, ensure that the scrolling indicator does not appear on the same axis to prevent user confusion caused by redundant controls.
