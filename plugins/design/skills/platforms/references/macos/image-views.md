# Image views
An image view presents either a static image or an animated sequence against a background that may be transparent or opaque

## Platform guidance — macOS
- **If your application requires an image view that allows editing, utilize an image well.** An **image well** is an image view capable of handling copying, pasting, dragging, and clearing its content using the Delete key.
- **To create a tappable image, employ an image button rather than a standard image view.** An **image button** holds an image or icon, displays within a view, and triggers an immediate action specific to the application.
