# Custom intents — overview
If your application facilitates an everyday task that falls outside of any SiriKit domain, you must create a custom intent to represent it (refer to [System intents](#System-intents) for the list of domains). Additionally, you can leverage a custom or system intent to support a Shortcut, allowing users to quickly trigger frequently performed actions using a simple phrase or by accepting a suggestion from Siri. To understand how to integrate your intents with the system for user discovery and addition to Siri, consult [Shortcuts and suggestions](#Shortcuts-and-suggestions).
