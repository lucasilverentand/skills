# Customizing your App Clip Code
Use either [App Store Connect](https://appstoreconnect.apple.com) or the [App Clip Code Generator](https://developer.apple.com/app-clips/resources/) command-line utility to produce App Clip Codes, adhering strictly to best practices for reliable scanning.

- **Always utilize the generated App Clip Code.** Do not attempt to design your own version or modify a code that has been generated. Filters, color augmentations, glows, shadows, gradients, or reflections must not be applied, as these elements degrade the scanning experience. If scaling a generated App Clip Code, maintain its original aspect ratio and ensure that all attributes, such as stroke widths, are scaled proportionally.
- **Select colors providing sufficient contrast for accurate scanning.** Each App Clip Code employs three colors: a foreground color, a background color, and a third color automatically derived from the first two. Both [App Store Connect](https://appstoreconnect.apple.com) and the [App Clip Code Generator](https://developer.apple.com/app-clips/resources/) command-line tool provide predefined color pairs, or you may select custom foreground and background colors. Be advised that choices leading to a suboptimal scanning experience are unsupported. If your color combination is unsuitable, neither App Store Connect nor the command-line tool will generate a valid App Clip Code. To assist in selecting an effective color scheme, both tools include features that suggest alternative foreground colors based on your chosen background color. For further details, consult [Creating App Clip Codes with the App Clip Code Generator](apple:AppClip/creating-app-clip-codes-with-the-app-clip-code-generator) and [Creating App Clip Codes with App Store Connect](apple:AppClip/creating-app-clip-codes-with-app-store-connect).
