# Activity rings
You can enhance your app’s health and wellness offerings by displaying the Activity ring element to illustrate users' advancement toward their Move, Exercise, and Stand objectives. Since the Activity app dictates the position and color of each ring, users are already familiar with this element and its meaning.

- **Use Activity rings exclusively for Move, Exercise, and Stand data.** Activity rings consistently represent progress in these specific areas. Do not attempt to repurpose or modify Activity rings for other data types or purposes. Never display Move, Exercise, or Stand progress using a ring-like element that is not an Activity ring.
- **Use Activity rings to display the progress of a single individual.** Never use Activity rings to represent data for more than one person; ensure clarity regarding whose progress is being shown by including a label, photo, or avatar.
- **Do not use Activity rings as ornamentation.** Activity rings serve to convey information; they are not merely decorative elements. Never display Activity rings within labels or background graphics.
- **Do not use Activity rings for branding.** Use Activity rings strictly to display Activity progress within your application. Never include them in your app’s icon or marketing materials.
- **Maintain Activity ring and background colors.** For a consistent user experience, the visual appearance of Activity rings must remain constant regardless of where they appear. Never alter the look of the rings or background by using filters, changing colors, or modifying opacity. Instead, design the surrounding interface to harmonize with the rings. For instance, enclose the rings within a circle. Always scale the rings appropriately so they do not appear disconnected or misplaced.
- **Maintain Activity ring margins.** An Activity ring element requires a minimum outer margin equal to the distance between rings. Never allow any other elements to crop, obstruct, or encroach upon this margin or the rings themselves. If you wish to display an Activity ring element within a circle, adjust the corner radius of the enclosing view rather than applying a circular mask.
- **Differentiate other ring-like elements from Activity rings.** Mixing different ring styles can result in a visually confusing interface. If you must include other rings, use padding, lines, or labels to separate them from Activity rings. Color and scale can also assist in providing visual distinction.
- **Provide app-specific information only within Activity notifications.** The system already delivers updates regarding Move, Exercise, and Stand progress. Do not repeat this information, nor should you display an Activity ring element in your app’s notifications. While referencing Activity progress in a notification is acceptable, it must be unique to your app and should not replicate the data provided by the system.

For developer guidance, see [HKActivityRingView](apple:HealthKitUI/HKActivityRingView).
