# HealthKit — full guidelines — overview
> **Important**
> Your application must not request access to users' private health data unless it provides relevant health and fitness functionality.

For instance, a nutrition application may need permission to retrieve users' activity levels and weight data in order to suggest dietary recommendations and set calorie goals. In this context, the nutrition app can also contribute data—such as logged caloric intake—to HealthKit, allowing it to be incorporated into global progress metrics.

For developer resources, consult [HealthKit](apple:HealthKit).
