# Inputs
Overview of Application/Game Control and Data Input Mechanisms

## Topics
- **Action button** — This feature provides users with rapid access to preferred functions on compatible iPhone and Apple Watch models.
- **Apple Pencil and Scribble** — Beyond functioning as a pointer and UI interaction tool, Apple Pencil enables drawing, handwriting, and annotation to be performed naturally and effortlessly.
- **Camera Control** — Camera Control offers direct management of the application's camera experience.
- **Digital Crown** — The Digital Crown serves as a critical hardware input mechanism for both Apple Vision Pro and Apple Watch.
- **Eyes** — In visionOS, users direct their gaze toward a virtual object to designate it as an interactive target.
- **Focus and selection** — Focus assists users in visually confirming the object they intend to interact with.
- **Game controls** — Precise and intuitive game controls improve gameplay quality and deepen player immersion.
- **Gestures** — A gesture is a physical movement used by a person to directly influence an object within an application or game on their device.
- **Gyroscope and accelerometer** — Onboard gyroscopes and accelerometers supply data regarding the device's movement within the physical environment.
- **Keyboards** — A physical keyboard functions as an essential input device for tasks such as text entry, game play, and application control.
- **Nearby interactions** — Nearby interactions enable on-device experiences that incorporate the presence of people and objects in the immediate surroundings.
- **Pointing devices** — Users can employ pointing hardware, such as a trackpad or mouse, to navigate the interface and trigger actions.
- **Remotes** — The Siri Remote is the primary input method for Apple TV, allowing users to maintain a connection with onscreen content from across the room.
