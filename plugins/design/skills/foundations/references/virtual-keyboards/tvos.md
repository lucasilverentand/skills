# Virtual keyboards
On devices lacking a physical keyboard, the operating system provides several virtual keyboard options for data input

## Platform guidance — tvOS
tvOS presents a linear virtual keyboard when users select a text field using the Siri Remote.

> **Note**
> When devices other than the Siri Remote are utilized, a grid keyboard screen appears, and the content layout automatically adjusts to suit the keyboard.

When a digit entry view is activated, tvOS displays a specialized keyboard for numerical input. For further instructions, consult **Digit entry views**.
