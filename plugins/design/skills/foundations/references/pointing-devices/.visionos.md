# Pointing devices
Users can utilize a pointing device, such as a trackpad or mouse, to navigate the interface and initiate actions

## Platform guidance — visionOS
In visionOS, users can integrate an external keyboard or pointing device and utilize both inputs concurrently with their eyes and hands. If the user focuses on a specific element and subsequently moves the pointer, the system shifts focus to whatever is beneath the pointer. Your application does not require any specific implementation to support this functionality.

When an external pointing device is connected, the user's gaze dictates the pointer's current context. For instance, if a user shifts their focus from one window to another, the pointer's context transitions smoothly to the new window.

If a gesture-capable pointing device (such as a trackpad or mouse) is employed, the pointer automatically conceals itself during gestures to reduce visual clutter. In this scenario, the pointer remains hidden until it is moved, at which point it reappears in the location the user is currently viewing.
