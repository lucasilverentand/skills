# Text alignment
- **Adjust text alignment to match the interface direction, if the system doesn’t do so automatically.** If you have left-aligned text content in a left-to-right (LTR) environment, for instance, you must right-align that text to maintain its mirrored position within a right-to-left (RTL) context.
- **Align a paragraph based on its language, not on the current context.** If a paragraph—defined as having three or more lines of text—is misaligned relative to its inherent language, readability suffers. For example, right-aligning text that is inherently LTR can make the start of every line difficult to perceive. To enhance readability, maintain alignment consistency for one- and two-line text blocks according to the current context's reading direction, but align multi-line paragraphs based on their native language. **Use a consistent alignment for all text items in a list.** To guarantee an optimal scanning and reading experience, ensure that every item within a list maintains uniform alignment, even if those items are displayed using a different script.
