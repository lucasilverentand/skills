# Inline
Inline designs encompass both utilitarian small and large formats.

Utilitarian small layouts are intended to occupy a rectangular space in the corner of a watch face, such as those found on Chronograph and Simple watch faces. Supported content types include an image, interface icon, or circular graph.

When designing visuals for a utilitarian small layout, use the following dimensions as guidance:

|Content|38mm|40mm/42mm|41mm|44mm|45mm/49mm|
|---|---|---|---|---|---|
|Flat|9-21x9 pt (18-42x18 px @2x)|10-22x10 pt (20-44x20 px @2x)|10.5-23.5x21 pt (21-47x21 @2x)|N/A|12-26x12 pt (24-52x24 px @2x)|
|Ring|14x14 pt (28x28 px @2x)|14x14 pt (28x28 px @2x)|15x15 pt (30x30 px @2x)|16x16 pt (32x32 px @2x)|16.5x16.5 pt (33x33 px @2x)|
|Square|20x20 pt (40x40 px @2x)|22x22 pt (44x44 px @2x)|23.5x23.5 pt (47x47 px @2x)|25x25 pt (50x50 px @2x)|26x26 pt (52x52 px @2x)|

The utilitarian large layout is primarily text-based, but it also supports an interface icon placed on the leading side of the text. This layout spans the bottom section of a watch face, similar to Utility or Motion watch faces.

For designing visuals intended for a utilitarian large layout, consult the following dimensions:

|Content|38mm|40mm/42mm|41mm|44mm|45mm/49mm|
|---|---|---|---|---|---|
|Flat|9-21x9 pt (18-42x18 px @2x)|10-22x10 pt (20-44x20 px @2x)|10.5-23.5x10.5 pt (21-47x21 px @2x)|N/A|12-26x12 pt (24-52x24 px @2x)|
