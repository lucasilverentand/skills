# Displaying App Clip Codes
When initiating the design of your App Clip Codes, select the variant that best suits how users will interact with your App Clip. If physical access to the App Clip Code is possible, utilize the NFC-integrated variant. Examples include:

- On a tabletop at a restaurant
- Near a register in a retail setting
- In a storefront window display
- On signage
- Printed on a gift card or coupon

If the App Clip Code must be placed in an area that is physically unreachable, or if it requires digital display, use a scan-only App Clip Code. Examples include:

- On posters or printed advertisements
- On signage located behind a counter or otherwise inaccessible in a storefront
- In digital formats such as emails, on social media posts, or on digital displays

Regardless of which variant you choose, it is crucial to carefully select the placement location for your App Clip Code to guarantee a successful scanning experience.

- **Include the App Clip logo when there is sufficient space.** The logo aids in identifying that the code launches an App Clip; however, if clear space cannot be met, use the App Clip Code design without the logo. Additionally, omit the App Clip logo if the code is placed on disposable paper or plastic items, or on materials associated with gambling or drinking. For instance, use the App Clip Code without the logo on playing cards, poker chips, or bar coasters. The App Clip logo is always part of the badge design located beneath the App Clip Code; never use the App Clip logo in isolation.
- **Place your App Clip Code exclusively on a flat or cylindrical surface.** If you place the App Clip Code on a cylindrical object—for example, a scooter’s handlebar—ensure the code's width does not exceed one-sixth of the cylinder’s circumference.
- **Help your App Clip Code maintain maximum flatness for easy scanning.** To optimize the scanning experience, avoid displaying App Clip Codes on pliable materials that easily fold or crumple, such as paper, plastic, or fabric. If you must make the App Clip Code available on a bag, flexible box, or other deformable item, display it on a rigid substrate—such as a card—that is attached to the object. If you create an App Clip Code sticker, ensure it adheres securely to flat surfaces.
- **Place your App Clip Code in a location that promotes reliable scanning.** For example, position a scan-only App Clip Code where there is adequate lighting for dependable scanning and the user does not need to scan from a wide angle.
- **Ensure the App Clip Code remains completely unobstructed.** Do not overlay the App Clip Code with text, images, or logos. Never animate or dim the App Clip Code.
- **Display the App Clip Code in an upright orientation.** Do not rotate the generated App Clip Code or present the center glyph at an angle. **Do not create App Clip Codes that are too small.** App Clip Codes must meet the following specifications.

|Type|Minimum size|
|---|---|
|Printed communications|Minimum diameter of 3/4 inch (1.9 cm).|
|Digital communications|Minimum size of 256×256 px. Use a PNG or SVG file.|
|NFC-integrated App Clip Code|The embedded NFC tag needs to be at least 35 mm in diameter or of equivalent size. For example, if your embedded NFC tag is 35 mm in diameter, your printed App Clip Code needs to be at least 1.37 inches (3.48 cm) in diameter.|

When determining the dimensions of your App Clip Codes, consider a distance to code size ratio of no more than 20:1. If possible, aim for a 10:1 ratio to ensure dependable scanning. For example, an App Clip scanned from 40 inches (101 cm) away requires a minimum diameter of 4 inches (10.16 cm).

If you display an App Clip Code near a QR Code or any other scannable item, select a size for the App Clip Code that is at least equivalent to the other code’s or item’s size.

**Provide sufficient clear space between an App Clip Code and adjacent graphics, codes, or materials.** The minimum required clear space around an App Clip Code must be equal to the distance between its center glyph and the circular code boundary. If you place your App Clip Code alongside another App Clip Code or machine-readable code, allow enough clear space to permit reliable scanning of each respective code.
