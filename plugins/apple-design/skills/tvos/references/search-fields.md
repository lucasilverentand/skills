# Search fields
A search field allows users to locate specific content within a collection using entered terms

## Platform guidance — tvOS
A search screen functions as a specialized keyboard interface, enabling users to enter search queries while displaying customizable results beneath the keyboard. For technical guidance on implementation, refer to [UISearchController](apple:UIKit/UISearchController).

**Provide suggestions to make searching easier.** Given that users on tvOS typically minimize typing, improve the search flow by offering popular and context-specific suggestions. This enhancement should also include recent searches when they are available. For guidance on implementing suggested searches with a search controller, see [Using suggested searches with a search controller](apple:UIKit/using-suggested-searches-with-a-search-controller).
