# Format menu
The Format menu enables users to modify text formatting attributes within the active document or container. If your application does not support rich text editing, you may omit this menu.

The Format menu typically includes the following top-level items, presented in this order:

|Menu item|Action|
|---|---|
|Font|Presents a submenu allowing adjustments to the font attributes of the selected text. Standard options include: Show Fonts, Bold, Italic, Underline, Bigger, Smaller, Show Colors, Copy Style, and Paste Style.|
|Text|Presents a submenu allowing adjustments to the text attributes of the selected text. Standard options include: Align Left, Align Center, Justify, Align Right, Writing Direction, Show Ruler, Copy Ruler, and Paste Ruler.|
