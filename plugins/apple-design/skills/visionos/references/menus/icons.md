# Icons
- **Represent menu item actions with familiar icons.** Icons assist users in recognizing common functions throughout the application. When representing standard operations such as Copy, Share, or Delete, employ system-level icons wherever they appear. For a catalog of symbols representing common actions, consult [Standard icons](icons.md#Standard-icons).
- **Don’t display an icon if you can’t find one that clearly represents the menu item.** Not all menu options require an icon. When adding icons to custom menu items, proceed cautiously to avoid confusing users with existing actions; do not include symbols solely for decorative purposes. **Use a single icon to introduce a group of similar items.** Instead of assigning individual icons for every action, or reusing the same icon across all entries, establish a unifying theme using the symbol for the first item. The menu item text must then maintain distinction among the remaining options.
