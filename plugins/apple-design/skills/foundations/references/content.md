# Content

## Topics
- **Charts** — Organize data visually into a chart to clearly and appealingly communicate information.
- **Image views** — An image view presents a single image, or occasionally an animated sequence, against either a transparent or opaque background.
- **Text views** — A text view renders multi-line, styled content that has the option of being editable.
- [Web views](web-views.md) — A web view loads and renders rich web content, including websites or embedded HTML, directly within the application.
