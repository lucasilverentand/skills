# Speech
Apple’s accessibility features enable individuals with speech impairments and those who prefer text-based interaction to communicate effectively using their devices.

- **Allow users to navigate and interact solely via the keyboard.** Users can activate Full Keyboard Access to control apps using a physical keyboard. The operating system also includes different accessibility shortcuts and numerous other [keyboard shortcuts](https://support.apple.com/en-us/102650) that are frequently utilized. You must refrain from overriding system-defined keyboard shortcuts and thoroughly test your application to confirm compatibility with Full Keyboard Access. Refer to **Keyboards** for more information, and consult [Support Full Keyboard Access in your iOS app](https://developer.apple.com/videos/play/wwdc2021/10120) for developer guidance.
- **Implement Switch Control support.** Switch Control is an assistive technology that allows users to operate their devices using external hardware, game controllers, or auditory cues like a click or pop. If your app or game supports navigation via Switch Control, users can execute actions such as selecting, tapping, typing, and drawing. For developer resources, see [Switch Control](apple:Accessibility/switch-control).
