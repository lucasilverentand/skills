# Standard keyboard shortcuts
**Generally, avoid repurposing standard keyboard shortcuts for custom functions.** Users may become confused if the actions of familiar shortcuts differ within your application or game. Only consider redefining a standard shortcut if its intended action does not align with the user experience. For instance, if your application lacks text editing capabilities, it may not require a text-styling command like Italic; in such cases, repurposing `Command–I` for a more relevant action, such as Get Info, might be appropriate.

Users expect the following standard keyboard shortcuts to perform the actions detailed in the table below.

The system also provides several keyboard shortcuts for use with localized versions of the operating system, specific keyboards, layouts, and input methods. These shortcuts do not correspond directly to menu commands.

|Keyboard shortcut|Action|
|---|---|
|Control-Space|Toggles between the current and the previously used input source.|
|Control-Option-Space|Switches to the next input source in the sequence.|
|[Modifier key]-Command-Space|Function varies based on context.|
|Command-Right arrow|Changes the keyboard layout to the current Roman script layout.|
|Command-Left arrow|Changes the keyboard layout to the current system script layout.|
