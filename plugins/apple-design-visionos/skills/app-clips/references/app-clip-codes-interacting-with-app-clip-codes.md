# Interacting with App Clip Codes
App Clip Codes come in two types: *scan-only*, or those incorporating an NFC tag (*NFC-integrated*).

The scan-only version features a camera icon centrally, indicating that users should employ the Camera app or the Code Scanner within Control Center to scan the App Clip Code. Conversely, the NFC-integrated version displays an iPhone icon at its center, prompting users to hold their device near the App Clip Code or scan it using the NFC Tag Reader in Control Center. Note that an NFC-integrated App Clip Code can also be scanned using the Camera app or the Code Scanner in Control Center. For instance:

- A coffee shop could feature an App Clip Code on its menu. A patron could hold their device near the code to instantly launch the shop’s App Clip and place an order.
- A gas station might affix an NFC-integrated App Clip Code to each pump. A traveler could hold their device close to the code to launch the station’s App Clip and settle payment for their refueling.
- A video game developer could distribute marketing materials containing an App Clip Code at an industry event. An attendee could scan the code to launch an App Clip serving as a playable demonstration of their newest video game.
