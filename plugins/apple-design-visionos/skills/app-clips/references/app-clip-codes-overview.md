# App Clip Codes — overview
App Clip Codes represent the most effective way for users to discover your App Clip. Their distinctive appearance ensures immediate recognition and provides a fast, secure mechanism for launching the App Clip.

All App Clip Codes must utilize Apple's provided designs and comply with established guidelines for size, placement, and printing. You have the option to create codes using a badge design that features the App Clip logo ( App Clip) or, when space is constrained, a version without it. Additionally, you may create codes using either a default color pairing or by specifying custom foreground and background colors. For detailed developer guidance, consult [Creating App Clip Codes](apple:AppClip/creating-app-clip-codes).
