# Using custom fonts
- **Ensure custom fonts are readable.** Users must be able to easily decipher your typeface across different viewing distances and under diverse conditions. When utilizing a custom font, follow the recommended minimum sizes for different styles and weights detailed in [Specifications](#Specifications).
- **Integrate accessibility features into custom fonts.** System fonts automatically support Dynamic Type (where available) and respond to user accessibility settings, such as Bold Text. If you employ a custom font, ensure it mirrors these behaviors. For developer guidance, consult [Applying custom fonts to text](apple:SwiftUI/Applying-Custom-Fonts-to-Text). In a Unity game, you may use [Apple’s Unity plug-ins](https://github.com/apple/unityplugins) to enable Dynamic Type support. If the plug-in is unsuitable for your game, provide alternative mechanisms allowing players to adjust text size.
