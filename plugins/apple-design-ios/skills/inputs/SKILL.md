# Inputs
Overview of Application/Game Control and Data Input Mechanisms

## When to use
- User asks about **inputs** on iOS and iPadOS (e.g. `"how do I design inputs for iPhone"`).
- User is building an iPhone UI that needs inputs and wants to follow Apple's guidelines.
- User asks to audit or review inputs in an iOS and iPadOS design.
- User mentions inputs in the context of an iPhone app, game, or interface.

## Topics
- [Action button](action-button.md) — This feature provides users with rapid access to preferred functions on compatible iPhone and Apple Watch models.
- [Apple Pencil and Scribble](apple-pencil-and-scribble.md) — Beyond functioning as a pointer and UI interaction tool, Apple Pencil enables drawing, handwriting, and annotation to be performed naturally and effortlessly.
- [Camera Control](camera-control.md) — Camera Control offers direct management of the application's camera experience.
- [Digital Crown](digital-crown.md) — The Digital Crown serves as a critical hardware input mechanism for both Apple Vision Pro and Apple Watch.
- [Eyes](eyes.md) — In visionOS, users direct their gaze toward a virtual object to designate it as an interactive target.
- [Focus and selection](focus-and-selection.md) — Focus assists users in visually confirming the object they intend to interact with.
- [Game controls](game-controls.md) — Precise and intuitive game controls improve gameplay quality and deepen player immersion.
- [Gestures](gestures.md) — A gesture is a physical movement used by a person to directly influence an object within an application or game on their device.
- [Gyroscope and accelerometer](gyro-and-accelerometer.md) — Onboard gyroscopes and accelerometers supply data regarding the device's movement within the physical environment.
- [Keyboards](keyboards.md) — A physical keyboard functions as an essential input device for tasks such as text entry, game play, and application control.
- [Nearby interactions](nearby-interactions.md) — Nearby interactions enable on-device experiences that incorporate the presence of people and objects in the immediate surroundings.
- [Pointing devices](pointing-devices.md) — Users can employ pointing hardware, such as a trackpad or mouse, to navigate the interface and trigger actions.
- [Remotes](remotes.md) — The Siri Remote is the primary input method for Apple TV, allowing users to maintain a connection with onscreen content from across the room.
