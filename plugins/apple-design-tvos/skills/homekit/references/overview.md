# HomeKit — overview
Your application (iOS, tvOS, or watchOS) can integrate with HomeKit—and consequently the Home app—to deliver a tailored or accessory-specific user experience. Examples include:

- Assisting users in setting up, naming, and organizing their accessories
- Enabling detailed configuration and control of the accessory
- Granting access to proprietary accessory features
- Demonstrating how to build robust, hands-free automations
- Offering technical support

For development guidance, consult [HomeKit](apple:HomeKit). If you hold MFi licensing, visit the [MFi portal](https://mfi.apple.com) for instructions regarding accessory packaging naming and messaging.
