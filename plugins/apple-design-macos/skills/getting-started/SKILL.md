# Getting started
Develop an application or game that delivers a cohesive, native experience across all supported platforms

## When to use
- User asks about **getting started** on macOS (e.g. `"how do I design getting started for Mac"`).
- User is building a Mac UI that needs getting started and wants to follow Apple's guidelines.
- User asks to audit or review getting started in a macOS design.
- User mentions getting started in the context of a Mac app, game, or interface.

## Topics
- [Designing for iOS](designing-for-ios.md) — Users rely on their iPhone as a constant companion for staying connected, playing games, consuming media, completing tasks, and tracking personal data anywhere.
- [Designing for iPadOS](designing-for-ipados.md) — Users value the combination of power, mobility, and adaptability of the iPad for enjoying media, playing games, and undertaking detailed productivity or creative projects.
- [Designing for macOS](designing-for-macos.md) — Users depend on the Mac's power, expansive canvas, and flexibility when performing intensive productivity tasks, viewing content, or gaming, often utilizing multiple applications simultaneously.
- [Designing for tvOS](designing-for-tvos.md) — Users appreciate the vivid content, immersive experiences, and simplified interactions that tvOS provides across media, gaming, fitness, education, and home companion applications.
- [Designing for visionOS](designing-for-visionos.md) — When users wear Apple Vision Pro, they enter an infinite 3D environment where your application or game can be experienced while remaining aware of the physical surroundings.
- [Designing for watchOS](designing-for-watchos.md) — When users check their Apple Watch, they know it provides immediate access to vital information and allows them to complete simple, time-sensitive tasks regardless of whether they are moving or still.
- [Designing for games](designing-for-games.md) — When users play your game on an Apple device, they immerse themselves in the world you created while benefiting from the platform's inherent features.
