# The menu bar — overview
Menu bar menus on iPad are comparable to those on Mac, appearing in the same sequence and featuring familiar menu items. By adopting the structure users expect from their Mac experience, you help them immediately understand and benefit from the iPad menu bar as well.

iPadOS keyboard shortcuts utilize the same patterns found in macOS. For reference, consult [Standard keyboard shortcuts](keyboards.md#Standard-keyboard-shortcuts).

Menus in the menu bar share most of the appearance and behavior characteristics common to all menu types. To learn about menus generally—and how to properly organize and label items—see [Menus](menus.md).
