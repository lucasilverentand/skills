# Displaying a website icon
different websites include an icon that appears when bookmarked, within URL fields, or on a device's Home screen. For websites that integrate Apple Pay, this icon is displayed in both the summary view and the payment sheet of the device authorizing the transaction. This visual element confirms to the user that the payment is being routed correctly.

If your website supports Apple Pay, you must supply an icon in the following dimensions for use within the payment sheet and summary view:

|@2x|@3x|
|---|---|
|60x60 pt (120x120 px @2x)|60x60 pt (180x180 px @3x)|
