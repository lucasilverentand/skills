# Charting data
Presenting data through a chart enhances the clarity and appeal of your communication

## When to use
- User asks about **charting data** on macOS (e.g. `"how do I design charting data for Mac"`).

## Quick principles
- **Use a chart when you want to highlight important information about a dataset** — Since charts are highly visible, they naturally capture user attention
- **Keep a chart simple, letting people choose when they want additional details** — Avoid the urge to include every possible data point in a single chart
