# System experiences

## When to use
- User asks about **system experiences** on watchOS (e.g. `"how do I design system experiences for Apple Watch"`).
- User is building an Apple Watch UI that needs system experiences and wants to follow Apple's guidelines.
- User asks to audit or review system experiences in a watchOS design.
- User mentions system experiences in the context of an Apple Watch app, game, or interface.
