# Menus and actions

## When to use
- User asks about **menus and actions** on watchOS (e.g. `"how do I design menus and actions for Apple Watch"`).
- User is building an Apple Watch UI that needs menus and actions and wants to follow Apple's guidelines.
- User asks to audit or review menus and actions in a watchOS design.
- User mentions menus and actions in the context of an Apple Watch app, game, or interface.

## Topics
- [Activity views](activity-views.md) — An activity view, often referred to as a *share sheet*, presents different tasks available within the current context.
- [Buttons](buttons.md) — A button triggers an immediate action upon interaction.
- [Context menus](context-menus.md) — A context menu provides access to functionality directly associated with an item, without cluttering the interface.
- [Dock menus](dock-menus.md) — On macOS, a secondary click on an application or game icon in the Dock reveals a Dock menu that includes both system-provided and custom items.
- [Edit menus](edit-menus.md) — An edit menu allows users to make changes to selected content in the current view, while also offering related commands such as Copy, Select, Translate, and Look Up.
- [Home Screen quick actions](home-screen-quick-actions.md) — Home Screen quick actions allow users to perform app-specific tasks directly from the Home Screen.
- [Menus](menus.md) — A menu displays its options when interacted with, offering a space-efficient way to present commands within your app or game.
- [Ornaments](ornaments.md) — In visionOS, an ornament presents controls and information related to a window without crowding or obscuring its contents.
- [Pop-up buttons](pop-up-buttons.md) — A pop-up button displays a menu containing options that are mutually exclusive.
- [Pull-down buttons](pull-down-buttons.md) — A pull-down button reveals a menu of items or actions that are directly related to the button's intended purpose.
- [The menu bar](the-menu-bar.md) — On a Mac or iPad, the menu bar at the top of the screen displays the application's or game's top-level menus.
- [Toolbars](toolbars.md) — A toolbar provides convenient access to frequently used commands, controls, navigation features, and search.
