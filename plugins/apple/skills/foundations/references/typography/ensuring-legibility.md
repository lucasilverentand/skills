# Ensuring legibility
**Select font sizes optimized for readability.** Your content must be readable across different viewing distances and environmental conditions. To ensure legibility on all devices, adhere to the recommended default and minimum text sizes for each platform, regardless of whether you use custom or system fonts. Remember that font weight also affects readability; if using a thin-weighted custom font, use sizes larger than those recommended to improve legibility.

|Platform|Default size|Minimum size|
|---|---|---|
|iOS, iPadOS|17 pt|11 pt|
|macOS|13 pt|10 pt|
|tvOS|29 pt|23 pt|
|visionOS|17 pt|12 pt|
|watchOS|16 pt|12 pt|

- **Validate legibility across different contexts.** For instance, if you are developing game text, you must test its readability on every platform your game supports. If testing reveals that certain text is hard to read, consider increasing the type size, improving contrast by adjusting text or background colors, or utilizing typefaces known for optimal legibility, such as the system fonts.
- **Generally, refrain from using light font weights.** For example, when utilizing system-provided fonts, favor Regular, Medium, Semibold, or Bold weights. Avoid Ultralight, Thin, and Light weights, as they can be difficult to discern, particularly when the text is small.
