# Navigation and search

## Topics
- **Path controls** — Displays the file system location of a selected file or directory.
- **Search fields** — Allows users to query a collection of content using specific input terms.
- **Sidebars** — A sidebar appears on the leading edge of a view, enabling navigation between sections in your application or game.
- **Tab bars** — Facilitates navigation among the top-level sections of your application.
- **Token fields** — A specific type of text field that converts entered text into *tokens*, which are easy to select and manipulate.
