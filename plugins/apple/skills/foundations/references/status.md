# Status

## Topics
- **Activity rings** — Activity rings visualize a user's daily advancement toward the Move, Exercise, and Stand objectives.
- **Gauges** — A gauge presents a precise numerical value within a defined range.
- **Progress indicators** — Progress indicators assure users that the application is operational, even when loading content or executing lengthy operations.
- [Rating indicators](rating-indicators.md) — A rating indicator uses a sequence of horizontally aligned graphical symbols—stars by default—to communicate a specific ranking.
