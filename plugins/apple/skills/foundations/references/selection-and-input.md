# Selection and input

## Topics
- **Color wells** — A color well allows users to modify the hue of text, shapes, guides, and other visual elements on screen.
- **Combo boxes** — A combo box integrates a text input area with a drop-down button into a single control.
- [Digit entry views](digit-entry-views.md) — A digit entry view occupies the entire screen, prompting users to input a sequence of numbers (such as a PIN) using a digit-specific keyboard.
- [Image wells](image-wells.md) — An image well is a version of an image view that allows for editing.
- **Pickers** — A picker presents one or more scrollable lists containing distinct values from which users can select.
- **Segmented controls** — A segmented control consists of a linear arrangement of two or more segments, each functioning as a button.
- **Sliders** — A slider is a horizontal track featuring a control, known as a thumb, that users manipulate between defined minimum and maximum values.
- **Steppers** — A stepper is a two-part control used by users to adjust an incremental value up or down.
- **Text fields** — A text field is a rectangular region where users input or modify short, specific pieces of text.
- **Toggles** — A toggle allows users to select between two opposing states (like on or off), using differing appearances to denote each state.
- **Virtual keyboards** — For devices lacking physical keyboards, the system provides different types of virtual keyboards for data entry.
