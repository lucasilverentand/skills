# Launching
A smooth launch experience enables users to begin using your app or game immediately

## Platform guidance — visionOS
**Consider launching in the Shared Space even if your application supports full immersion.** Utilizing a window within Shared Space allows you to offer users context regarding your app or game while it loads, and it also provides a control mechanism for the user to initiate the fully immersive experience. Generally, users prefer having agency over when they transition into a Full Space, particularly if they are concurrently running other applications in Shared Space. For detailed guidance, consult **Immersive experiences**.
