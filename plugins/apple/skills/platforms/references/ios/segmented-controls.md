# Segmented controls
A segmented control presents as a linear arrangement of two or more segments, each acting as a button

## Platform guidance — iOS & iPadOS
A segmented control is appropriate when switching between closely related subviews. This component allows users to quickly navigate between linked views; for example, the segmented control within Calendar’s New Event sheet facilitates switching between creating a new event and a new reminder. However, if the application sections are entirely separate, you must utilize [Tab bars](tab-bars.md) instead.
